== Japanese Sake Dashboard

## About
The Japanese Sake Dashboard automatically fetches Japanese sake-related tweets and data from Twitter. It then displays the information in via a timeline and chart.

## Technologies
* Rails
* ChartJS
* Oauth
* Twitter API
* rSpec
* Bootstrap
* Ruby 2.2.1
